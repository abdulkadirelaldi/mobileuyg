# Assets Klasörü

Bu klasör uygulama icon'ları ve splash screen görsellerini içerir.

## Gerekli Dosyalar

- `icon.png` - Uygulama ikonu (1024x1024 px önerilir)
- `splash.png` - Splash screen görseli (1242x2436 px önerilir)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Not

Şu anda bu dosyalar placeholder olarak kullanılabilir. Gerçek görseller eklendiğinde bu dosyalar değiştirilebilir.

Expo, bu dosyalar olmadan da çalışabilir ancak uyarı verebilir.
